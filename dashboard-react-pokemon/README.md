# Pokemon Dashboard React

Dashboard interactivo para explorar Pokemon usando PokeAPI.

## Demo en vivo
https://dashboard-react-pokemon-esoq.vercel.app

## Descripcion
Aplicacion construida con React que consume la PokeAPI para mostrar informacion de Pokemon. Permite buscar por nombre o ID, ver una lista con paginacion y detalles de cada Pokemon.

## Tecnologias
- React 18
- PokeAPI
- Vite (o Create React App si usas eso)
- CSS (o Tailwind, o lo que uses)

## Caracteristicas
- Busqueda de Pokemon por nombre o ID
- Lista con paginacion e imagenes
- Detalles con estadisticas y tipos
- Funciona bien en movil, tablet y computadora

## Como instalar

npm install
npm run dev
